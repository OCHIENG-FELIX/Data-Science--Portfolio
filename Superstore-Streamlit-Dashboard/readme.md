# Superstore Business Intelligence Dashboard

**Interactive Web Application**

A live, interactive dashboard for analyzing Superstore retail data. Users can filter by region, category, and segment to explore sales trends, profit performance, RFM customer segmentation, and top performers.

### Features
- Real-time filtering (Region, Category, Segment)
- Key Performance Indicators (KPIs)
- Interactive charts (Trends, Regional Performance, Category Analysis)
- RFM Customer Segmentation
- Download filtered data as CSV
